<h1>NTFS (New Technology File System)</h1>

<hgroup>
<p><em>NTFS (New Technology File System )</em> é um tipo de sistema de arquivos padrão do
<em>WindowsNT (Windows New Technology)</em> criado pela Microsoft em 1993. Devido  ser mais 
confiavel e ter uma desepenho melhor em relação aos outros sistema de arquivos anteriores
fez com que fosse adotado nos sistemas operacionais posteriores da <b>Microsoft, 
como Windows XP, Windows Vista, Windows 7 e Windows Server 2008.</b>
</p>
<p>
O surgimento do Sistema de Arquivos <em>NTFS</em> surgiu pela a necessidade da <b>Microsoft de Bill Gates</b> querer se apoderar de uma 
fatia do mercado dominado pela <em>Unix</em> em aplicações de servidores. Até então a Microsoft tinha suas aplicações voltadas para uso 
doméstico e/ou escritório. Em cima dessa necessidade criou ou se o <b>WindowsNT</b> junto com ele veio o <em>Sistema de Arquivos NTFS.</em>
</p>
<img alt='NTFS' border='0' src='https://image.ibb.co/dgQCCp/NTFS.png'/>

<h2>Prós NTFS (New Techonology File System)</h2>
<p>
 &#10004;<b>Recuperação</b> - em caso de falhas, como o desligamento repentino do computador, 
o <em>NTFS</em> é capaz de reverter os dados à condição anterior ao incidente. 
Isso é possível, em parte, porque, durante o processo de boot, o sistema operacional 
consulta um arquivo de log que registra todas as operações efetuadas e entra em ação ao 
identificar nele os pontos problemáticos.
</p>
<p>
 &#10004;<b>Permissões de acesso</b> - O <em>NTFS</em>  é capaz de permitir que o usuário defina quem pode e como acessar pastas ou arquivos.
</p>
  <img src='https://image.ibb.co/e6SPje/39872636_299912290799544_7173803592980103168_n.jpg'/>
