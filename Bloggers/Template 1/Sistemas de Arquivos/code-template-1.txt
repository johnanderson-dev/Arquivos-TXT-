  <h1>NTFS (New Technology File System)</h1>

<hgroup>
<p>NTFS (New Technology File System ) é um tipo de sistema de arquivos padrão do
WindowsNT (Windows New Technology) criado pela Microsoft em 1993. Devido  ser mais 
confiavel e ter uma desepenho melhor em relação aos outros sistema de arquivos anteriores
fez com que fosse adotado nos sistemas operacionais posteriores da Microsoft, 
como Windows XP, Windows Vista, Windows 7 e Windows Server 2008.
</p>
<p>
O surgimento do Sistema de Arquivos NTFS surgiu pela a necessidade da Microsoft de Bill Gates querer se apoderar de uma fatia do mercado dominado pela Unix em aplicações de servidores. Até então a Microsoft tinha suas aplicações voltadas para uso doméstico e/ou escritório. Em cima dessa necessidade criou ou se o WindowsNT junto com ele veio o Sistema de Arquivos NTFS.
</p>
<img alt='NTFS' border='0' src='https://image.ibb.co/dgQCCp/NTFS.png'/>

<h2>Prós NTFS (New Techonology File System)</h2>
<p>
 &#10004;Recuperação - em caso de falhas, como o desligamento repentino do computador, 
o NTFS é capaz de reverter os dados à condição anterior ao incidente. 
Isso é possível, em parte, porque, durante o processo de boot, o sistema operacional 
consulta um arquivo de log que registra todas as operações efetuadas e entra em ação ao 
identificar nele os pontos problemáticos.
</p>
<p>
 &#10004;Permissões de acesso - O NTFS  é capaz de permitir que o usuário defina quem pode e como acessar pastas ou arquivos.
</p>
