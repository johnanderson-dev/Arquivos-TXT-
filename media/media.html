Explicação sobre Media Query:

  • Media Query: Refere-se a tela do dispositivo (algo fisico)
  • Media Type: Refere-se ao tipo de tela baseado na altura e largura; Portrait (celular em pé) ou Lansdcape (celular deitado)
  • Media Features: São trechos de condigos que irão impor condições de acordo com a largura e altura do dispositivo
OBS: O media type Portrait é baseado na largura width, seria a mesma coisa que colocar @media max-width; Se for do tipo Landscape será a mesma coisa que max-height(altura).
Exemplo:
  • Se colocar @media (max-width: 768px); O elemento vai assumir os atributos setados de 0px a 768px na largura;
  • Se colocar @media (min-width: 768px); O elemento vai assumir os atributos setados de 768px para cima na largura;
  • Se colocar @media (max-height: 768px); O elemento vai assumir os atributos setados de 0px a 768px na Altura;
  • Se colocar @media (min-height: 768px); O elemento vai assumir os atributos setados de 768px a pra cima na Altura;

Operadores:

And

Usando AND irá ser incluido a soma de mais uma condição, media features no codigo:
EX: @media (max-width: 768px) and (orientation: lansdcape)
Como pode ver isso faz com que tenha duas condições. Mas quando se usa um max-width (Portrait) e orientation: lansdcape o max-width irá ser setado apenas quando atingir o valor dele e se for obtido menos ou mais não ira assumir os valores. E o lansdcape terá um valor a menos até 0 com os valores setados.
Ex: Se dentro dessa condição o body for da cor amarelo de acordo com a primeira condição só tera cor amarela quando atingir 768px, abaixo disso com 767 e acima 769 não irá setar.
A segunda condição só sera setada quano tiver um amenos na altura de acordo com que foi passado. Ou seja, se foi passado 768 px , só irá setar os valores de 767px pra baixo.

De 767 px de altura para baixo
Maior que 768 px
Isso ocorre por que segundo a segunda condição a orientação é pra Landscape, baseada na altura. Então o media query será em cima da altura
Usando agora duas condições de media features para aplicar as condições se forem validas
@media (min-width: 320px) and (max-width: 800px)
Nessa media query features informa que será aplicado a condição de 320 px pra cima e na outra informa que sera aplicado de 800 pra baixo. Ou seja, irá ficar entre os dois, de 320 à 800 px.
Se for menor que 320 px e maior que 800 px, essa frase sera informada!
Se estiver dentro das condições essa frase será informada!

OR

Com o operador OR, a regra será aplicada se uma OU outra condição for válida.
@media (max-width: 800px), (min-width: 900px)
Nessa condição os valores só serão setados se a tela for menor que 800 xp e maior que 900. Se estiver entre 800 e 900 os valores não serão setados
Menor que 800 ou maior que 900 px!
Entre 800 e 900 px.
